About the “mixed conversation” responses Sobre las respuestas “mezcladas” de la conversación

In the assistant UI, answers can be “attached” to the current session (conversation id). If you keep using the same session, the model may keep context and responses can look mixed across turns.

En la UI del asistente, las respuestas pueden quedar “pegadas” a la sesión actual (conversation id). Si sigues usando la misma sesión, el modelo mantiene contexto y las respuestas pueden verse mezcladas.

  • To reset cleanly: start a new conversation/session.
  • Quick fix: clear the chat history for that session if your UI supports it.
  • Why it happens: session memory/context is intentionally used to keep continuity.
  • Para reiniciar limpio: inicia una nueva conversación/sesión.
  • Arreglo rápido: borra el historial de ese chat si tu UI lo permite.
  • Por qué pasa: la memoria/contexto de sesión se usa para mantener continuidad.
